### 第9章 
#### 9.3 弹性盒子
语法:

```
.father{
display:-webkit-box;
 display:box;
}
.son1{
-webkit-box-flex:1
}
.son2{
width:500px;
}
.son3{
-webkit-box-flex:2
}
结果 有一个布局:中间固定宽度，剩下的宽度两边以1 2进行分配，
注意可以对father进行设置margin
```
- 失效：多列中的column-\*属性对于盒子子元素无效
- float和clear对于弹性子元素无效，使用float会导致display属性计算为block。
- vertical-align对于弹性盒子的对其无效
